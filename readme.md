### 用户信息记录程序

---

#### 该程序为练习golang用的

#### 使用
切换到根目录下

go build ./src/main/main.go

main.exe或者main 直接运行

看参数请使用 main -h查看

-create 创建用户，需要搭配-username -gender -age 三个参数

-delete 删除用户，需要搭配-id

-update 修改用户，需要搭配-username -gender -age 三个参数

-list 列出所有用户，不需要搭配参数
